iOS 输入限制之 InputKit

深入 InputKit 框架，探索 iOS 文本输入的底层机制与输入限制的实现方式。

InputKit 是 iOS 中处理文本输入的核心框架。它连接了应用、系统输入设备和候选词系统，是理解输入体验的重要入口。

核心类与协议

InputKit 提供了一组类与协议来管理输入会话、文本处理与候选词。

Swift复制
import InputKit

class MyInputController: UIInputViewController, IKTextInput {
    override func textWillChange(_ textInput: UITextInput?) {
        // 在文本变化前处理
    }

    override func textDidChange(_ textInput: UITextInput?) {
        // 应用输入限制
    }
}
