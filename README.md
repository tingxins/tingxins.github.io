# tingxins.github.io

Personal blog for <https://midaigc.com>, migrated from Jekyll to Astro.

## Development

```bash
npm install
npm run dev
```

## Validation

```bash
npm run check
npm run build
npm run preview
```

## Deployment

GitHub Pages is deployed by `.github/workflows/deploy.yml` from the `master` branch.

## Legacy Jekyll

The previous Jekyll implementation is kept under `legacy-jekyll/` for reference only. Astro content now lives in `src/content/posts/`, and static files are served from `public/`.
